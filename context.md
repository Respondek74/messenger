# Ekornes UX Discovery — Project Context

## Files in this folder
- Ekornes_UX_Discovery_Index.html — Landing page, links to all views
- Ekornes_Expert_Review.html — CX analysis with pain points & recommendations
- Ekornes_Customer_Journey.html — 7-phase interactive journey narrative
- Ekornes_Service_Blueprint.html — Full service blueprint (actors, systems, processes)

## Tech stack
- Standalone HTML files, React via CDN (createElement, no JSX/Babel)
- Stressless SVG logo embedded as base64 data URI
- Google Fonts: Cormorant Garamond + DM Sans
- All text in English

## Design principles
- Scandinavian/luxury aesthetic, muted earth tones
- Unified light theme across all pages: #FAF8F5 background, #F5F2EF surface, white cards
- Fonts: Cormorant Garamond (headings, 300/400) + DM Sans (body, 300–600) + DM Mono (Blueprint technical labels)
- Text: #1A1A1A primary, #555 body, #888/#999 secondary, #AAA/#BBB muted
- Borders: #E8E4DF throughout
- Accent colors: Gold #C9A84C (Expert Review), Phase colors (Journey/Blueprint), Pain #c0392b, Opportunity #2d7a4f
- Standardised header: back-link left, title in Cormorant Garamond, logo right (56px, opacity 0.85)
- Standardised footer: "SYBIT GMBH — DIVISION CREATION — EKORNES CONCEPT 2026" in #BBB
- All pages interlinked, sticky labels on horizontal scroll views
